# Compute Commons

The next AI divide isn't intelligence. It's access.

A peer-governed network where verified contribution to AI earns redeemable compute access — not cash, access.

## Live
https://computecommons.io

## Stack
- Next.js 15 / React 18 / TypeScript
- Zustand + TanStack Query
- Tailwind CSS
- Axios

## Routes
- / — Homepage
- /dashboard/contributor — Contributor Dashboard
- /dashboard/validator — Validator Dashboard
- /dashboard/governance — Governance
- /dashboard/trust — Trust Scores
- /leaderboard — Leaderboard

## Backend
FastAPI + Python — backend integration in progress.
API base URL configured via NEXT_PUBLIC_API_URL environment variable.

## Development
npm install
npm run dev
